# Quantum Playground

My journey learning quantum computing with Qiskit. Includes basic circuits, key algorithms, and a bonus QAOA for optimization tied to sales scenarios.

## Setup
1. Clone repo: `git clone https://github.com/austinpaluch/quantum-playground.git`
2. Create venv: `python3 -m venv qiskit-env`
3. Activate: `source qiskit-env/bin/activate`
4. Install: `pip install -r requirements.txt`
5. Open in VS Code and run notebooks.

## Notebooks
- **01_basic_circuits.ipynb**: Hello World, superposition, Bell states.
- **02_key_algorithms.ipynb**: Deutsch-Jozsa, Grover's, Teleportation.
- **03_qaoa_optimization.ipynb**: QAOA for toy sales optimization.

## What I Learned
- Quantum basics: Superposition, entanglement.
- Algorithms: Speedups for search and oracles.
- Application: Linking to real-world optimization from my sales background.

Visuals: Circuits and histograms included in notebooks.
